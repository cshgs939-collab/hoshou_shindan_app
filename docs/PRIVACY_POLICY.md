# プライバシーポリシー

**まもる計算**（以下「本アプリ」）のプライバシーポリシーです。

最終更新日: 2026年6月25日

---

## 1. 基本方針

本アプリは、ユーザーの入力データを**端末内のみ**で処理・保存し、**外部サーバーへ送信しません**。広告配信や第三者へのデータ販売も行いません。

## 2. 収集する情報

本アプリは、ユーザーが診断画面に入力した以下の情報を端末内に保存する場合があります。

- 年齢、年収、家族構成、雇用形態
- 住宅・ローン情報（持家/賃貸、残債、家賃、団信の有無など）
- 既存の保険・貯蓄の概算
- 診断結果（不足保障額、内訳）

これらはすべて**ユーザー自身の操作による入力**です。

## 3. 情報の利用目的

入力データは、以下の目的でのみ端末内で利用されます。

- 必要保障額の試算
- 診断履歴の表示・比較
- PDF / JSON エクスポート（ユーザー操作時）
- ホーム画面ウィジェットへの結果表示
- 再診断リマインダー通知（ユーザーが有効化した場合）

## 4. 外部送信・共有

- 本アプリは入力データを**自動的に外部へ送信しません**
- PDF / JSON の共有は、ユーザーが OS の共有機能を使った場合のみ行われます
- インターネット接続は、アプリの基本機能には不要です

## 5. データの保存場所と暗号化

- 診断履歴・設定は端末内の Hive データベースに保存されます
- AES 暗号化キーは端末のセキュアストレージ（Keychain / EncryptedSharedPreferences）で管理されます
- アプリをアンインストールすると、端末内データは削除されます

## 6. 通知

年次リマインダー通知は、ユーザーが設定画面で有効化した場合のみ送信されます。通知内容に個人を特定できる情報は含みません。

## 7. 第三者サービス

本アプリは、現時点で Firebase Analytics 等の第三者解析 SDK を組み込んでいません。

## 8. お子様の利用

本アプリは保険・家計の試算を目的としており、13歳未満の方を対象としていません。

## 9. ポリシーの変更

本ポリシーを変更する場合、アプリ内または GitHub リポジトリ上で更新日を明示します。
