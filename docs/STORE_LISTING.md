
## 詳しい説明

```
【まもる計算 — 家族を守る保障額シミュレーター】

年齢・年収・家族構成・住宅ローンから、万一の際に必要な保障額を端末内で簡易試算します。
保険の販売・勧誘は行わない、中立的なシミュレーションツールです。

■ 主な機能
・5分でわかる不足保障額の試算
・生活費・教育費・住居費・その他ローンの内訳表示
・持ち家/賃貸/実家、団信の有無に対応
・診断履歴の保存（最大20件・端末内暗号化）
・教育方針・雇用形態のシナリオ比較
・PDF / JSON エクスポート
・ホーム画面ウィジェット（不足額を表示）
・年1回の見直しリマインダー（任意）

■ こんな方におすすめ
・生命保険の保障額が足りているか確認したい
・住宅ローンと団信の関係を含めて試算したい
・ライフステージの変化に合わせて定期的に見直したい

■ プライバシー
入力データは端末内のみで処理され、外部サーバーへ送信されません。

■ 免責
本アプリの結果は簡易試算であり、実際の保障設計の根拠にはなりません。
保険商品のご加入にあたっては、専門家へのご相談をおすすめします。
```
