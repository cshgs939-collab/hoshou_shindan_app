# まもる計算 v2.0.0

家族構成・住宅ローンから必要保障額を試算し、不足額を可視化するアプリです。保険販売・勧誘ではなく、参考用の簡易診断ツールです。

## 配布ファイル

| ファイル | 用途 |
|------|------|
| `まもる計算_v2.0.0.apk` | Android 端末へ直接インストール（推奨） |
| `まもる計算_v1.0.0.apk` | 旧ビルド（起動クラッシュあり・非推奨） |
| `まもる計算_web_v1.0.0.zip` | Web 版（解凍後にブラウザで `index.html` を開く） |

## Android インストール

1. APK を Android 端末へ転送
2. 設定 → セキュリティ → 提供元不明のアプリを許可
3. APK をタップしてインストール

USB 接続時:

```bash
adb install -r まもる計算_v2.0.0.apk
```

## Web 版

```bash
cd /Users/maya2018/Downloads/hoshou_shindan_app
flutter run -d chrome
```

または:

```bash
./scripts/serve_web.sh
```

## 注意

- 試算結果は簡易計算であり、実際の保障設計の根拠にはなりません
- 端末内で完結し、入力データを外部送信しません
